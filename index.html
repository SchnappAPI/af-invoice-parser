1st Lake Invoice Tools

Water utility invoice processing — AppFolio integration
Tools
⬇️
AppFolio Invoice Downloader
Bulk downloads invoice attachments from AppFolio. Renames each file to its Reference value, downloads it to your computer, and exports a status CSV showing what was downloaded, skipped, or failed.
AF Invoice DL ← right-click → Bookmark This Link → Bookmarks Toolbar
🔍
Water Invoice Parser
Extracts structured data from water invoice PDFs and images using Claude AI. Load the DL status CSV to automatically match invoices, merge original AppFolio columns, and flag pay mismatches between Amount Due and Paid.
Open Parser
Workflow
End-to-end process
1
Export CSV from AppFolio — Accounting → Payable Invoices → Actions → Export as CSV. Make sure Txn ID and Reference columns are visible.
2
Run the DL bookmarklet on AppFolio — load the CSV in the panel, click Run Batch. Invoice PDFs download to your computer. Click Export Status CSV when done.
3
Open the Invoice Parser — drop the status CSV in the top zone, then drop all your downloaded invoice PDFs into the main zone.
4
Click Extract Data — Claude reads each invoice and pulls out all the billing fields. Pay mismatches are flagged in red automatically.
5
Export CSV or Excel — one file with all original AppFolio columns plus extracted invoice data side by side.
